# Ankur Public School 🎓

Welcome to the official repository of **Ankur Public School**. This is a modern, responsive website built to showcase the excellence and holistic development opportunities provided by our institution.

## 🏫 About Ankur Public School
Ankur Public School is dedicated to providing a "Knowledge is Power" environment since its inception. Our institution blends traditional values with modern pedagogy on a lush green 2-acre campus in Aligarh, Uttar Pradesh.

## ✨ Features
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop devices.
- **Dynamic Hero Section:** Engaging visuals showcasing school life and mission.
- **Comprehensive Sections:** About Us, Academics, Facilities, Gallery, and Contact.
- **Modern Tech Stack:** Built with React 19, Vite, and Tailwind CSS.
- **Smooth Animations:** Integrated with Framer Motion for a premium user experience.

## 🛠️ Tech Stack
- **Frontend:** React 19
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Animations:** Framer Motion

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ankur-public-school.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ankur-public-school
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
